# Prerequisites

In order to develop the application locally, the following tools are required:

- [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/)
- [`mise`](https://mise.jdx.dev/getting-started.html)

Mise handles installation and management of local development tools, runtimes and dependencies for this repository. Install them by running:

```shell
mise install
```
